**to_csv_all_stations.py**

(CAT)

**ONE STATION**

* Converteix un fitxer escopit per sbe data processing a csv (separat per comes) per poder-lo manipular fàcilment des de R i des de excel. 

* Cal especificar al final del fitxer la ruta del input file i la ruta del output file.

**MULTIPLE STATIONS**

* Fa el mateix que el "ONE STATION" però per tots els fitxers d'una carpeta.

* Només cas posar el input directory i el output dir.

* ATENCIÓ: no pot crear carpetes, s'ha de correr amb la carpeta de output ja creada

(ENG)

**ONE STATION**

* Convert a file outputted by SBE data processing to CSV (comma-separated) format for easy manipulation in both R and Excel.

* It's necessary to specify at the end of the file the path of the input file and the path of the output file.


**MULTIPLE STATIONS**

* Performs the same task as "ONE STATION" but for all files in a folder.

* Only provide the input directory and the output directory.

* CAUTION: it cannot create folders, it must be run with the output folder already created.
